# Infotech Assignment

A responsive landing page developed using HTML, CSS, and JavaScript based on the provided Figma design.

## Features

- Responsive design for Desktop, Tablet, and Mobile devices
- Modern UI matching the provided Figma design
- Smooth navigation and interactive elements
- Clean and organized code structure
- Cross-browser compatible

## Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## Project Structure

```
.
├── assets/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Rishabh2446/Infotech-assignment.git
```

2. Open the project folder

```bash
cd Infotech-assignment
```

3. Open `index.html` in your browser.
